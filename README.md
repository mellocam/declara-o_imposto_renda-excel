# declaracao_imposto_renda-excel
Sistema de Declaração de Imposto de Renda com Excel &amp; IA.

Descrição: o projeto automatiza a organização de dados para a declaração anual, separando titulares, informes de rendimentos e notas bancárias.

Funcionalidades:

    Gerenciamento de dados do titular.

    Consolidação de informes de múltiplos bancos (Stone, Original, etc.).

    Registro de entradas (Holerites) e integração de dados.
